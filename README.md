# Food Club
## API REST PHP


food_club_api é uma API REST escrita em PHP para fins educacionais.

## Features

- Cadastro de alunos, responsáveis e produtos.
- Compra de produtos e depósito

## Tech

food_club_api um projeto open-source para a gestão de rotas

- [simple-php-router](https://github.com/skipperbent/simple-php-router) - Gestor simples de rotas
- [composer](https://getcomposer.org/) - Gestor de pacotes PHP

## Installation

A instalação do ambiente foi feita utilizando [wamp](https://www.wampserver.com/en/).

Na pasta www do wamp, inicie um terminal e instale as dependências:

```sh
cd food_club_api
composer update
```
No arquivo food_club.sql estão disponíveis os scripts do banco de dados.
